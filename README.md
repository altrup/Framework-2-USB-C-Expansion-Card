# Framework 2 USB-C Expansion Card

A 2 USB-C expansion card for Framework laptops. Currently in design using the `TUSB8044A`. Will hopefully support USB 3.2, power delivery, and DP alt mode (through mux; MST hub ICs aren't widely available)
